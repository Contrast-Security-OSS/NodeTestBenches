# HapiTestBench
screener app for Hapi 18.

For Hapi 17 use: [hapi17](https://github.com/Contrast-Security-OSS/HapiTestBench/tree/hapi17)

## Installation
```bash
npm install
node server.js
```

## Requirements:
[mongodb](https://docs.mongodb.com/manual/installation/)
[mysql](https://www.mysql.com/)
